Redes

Las redes informáticas han permitido que personas de cualquier parte del mundo puedan intercambiar información de todo tipo entre ellas, solamente utilizando dispositivos que estén conectadas a internet, o a la misma red. La red consiste en un conjunto de sistemas informáticos independientes que están conectados entre sí, permitiendo así el intercambio de datos entre 2 o más dispositivos, para esto es necesario una conexión física y una conexión lógica, esta conexión lógica se establece a través de unos protocolos de red especiales, para esto está el TCP (Transmission Control Protocol). Dos dispositivos conectados entre sí pueden formar una red, estas se configuran con el objetivo de poder transmitir datos de un sistema a otro o de disponer recursos como servidores, bases de datos, entre otros.

Hay diferentes tipos de red, cada una con procesos y herramientas distintas para conectar dispositivos entre si, de las más comerciales y utilizadas encontramos
Personal Area Network (PAN)

esta permite realizar intercambio de datos en los dispositivos que se utilizan cotidianamente como smarthpones, ordenadores, tablas, entre otros. Esto se puede realizar a través de un cable, aunque las técnicas más utilizadas don la memoria USB o el conector FireWire.

Local Area Network (LAN).

Una red local de tales características puede incluir a dos ordenadores en una vivienda privada o a varios miles de dispositivos en una empresa. Asimismo, las redes en instituciones públicas como administraciones, colegios o universidades también son redes LAN. Un estándar muy frecuente para redes de área local por cable es Ethernet.

Global Area Network (GAN)

Una red global como Internet recibe el nombre de Global Area Network (GAN), sin embargo, no es la única red de ordenadores de esta índole. Las empresas que también son activas a nivel internacional mantienen redes aisladas que comprenden varias redes WAN y que logran, así, la comunicación entre los ordenadores de las empresas a nivel mundial

Wide Area Network (WAN)

la Wide Area Network (WAN) consiste en una red que comunica dispositivos en zonas geográficas como países o continentes, la diferencia es que Mientras que las redes LAN y las MAN pueden establecerse a causa de la cercanía geográfica del ordenador o red que se tiene que conectar en base a Ethernet, en el caso de las Wide Area Networks entran en juego técnicas como IP/MPLS (Multiprotocol Label Switching), PDH (Plesiochronous Digital Hierarchy), SDH (Synchronous Digital Hierarchy), SONET (Synchronous Optical Network), ATM (Asynchronous Transfer Mode) y, rara vez, el estándar obsoleto X.25.
